# 🍿 usePopcorn

A simple React movie explorer – search for any movie, rate it, and build your personal watchlist. Data is fetched from a movie API (no actual playback, just discovery and tracking).

✨ Features

· 🔍 Search movies – real‑time results from an external API
· ⭐ Rate movies – give your own score to each movie
· 📋 Watchlist – save movies you want to watch later
· 💾 Persistent data – ratings and watchlist are stored locally in your browser

🛠 Built with

· React (Hooks, Context API)
· React Router (optional)
· Fetch / Axios
· LocalStorage
